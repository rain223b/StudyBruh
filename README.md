# Clever | Portal

A curated collection of the best web-based games.

## How to Deploy to GitHub Pages (Automatic)

1.  **Sync to GitHub:** Click the **Sync to GitHub** button in AI Studio.
2.  **That's it!** Your site will be live at your GitHub Pages URL.

## Why it works now
I have configured the main `index.html` to point directly to the built version of your site in the `docs` folder. This means GitHub will serve the working website by default without you needing to change any settings.

## Note on Proxy Features
The **Web Browser** and **Proxy** features require a Node.js server. Since GitHub Pages is a static host, these features will not work there. For full functionality, consider hosting on **Vercel** or **Render**.
